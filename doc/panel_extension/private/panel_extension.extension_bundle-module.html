Module panel_extension.extension_bundle

extension_bundle module (c) 2005 travis f vachon

the extension_bundle module provides a Bundle class that wraps a GNOME panel extension bundle to provide easy access to files contained within the bundle. For more information, please see http://www.gnome.org/~tvachon
Classes
Bundle Bundle is a class to contain a bundle which provides methods to allow for easy extraction of files from the bundle.

Exceptions
InvalidBundle Used to indicate a bundle is either non-existent or is missing a necessary piece.
NoIconError Used to indicate that no icon was specified for this file.
