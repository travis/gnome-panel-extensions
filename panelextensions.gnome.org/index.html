We are proud to announce the release of GNOME panel extensions v0.0.2!

GNOME panel extensions are a new type of object for the GNOME panel. Like the current applet system, panel extensions are designed to be easy to write and integrate with GNOME. Unlike panel applets, however, panel extensions can be distributed and installed without the use of build tools or package managers, and should work on almost any recent installation of GNOME.

For more information on downloading and installing the panel extension system please see the Download section below. For a tutorial on writing extensions, see Writing Panel Extensions

Special thanks to Chris DiBona and everyone at the Google Summer of Code program who provided seed and grant money that made this project happen.

Also, eternal gratitude to Havoc, Jonathan, and everyone at GNOME who helped make this project a success.
